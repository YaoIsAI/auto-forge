# Forge 微信公众号宣传物料

## 文件说明

```
marketing/wechat/
├── copywriting.html          # 公众号文案（带一键复制功能）
├── cover.svg                 # 封面图（SVG 格式）
├── README.md                 # 说明文档
└── images/                   # 宣传图片
    ├── img1_hero.html        # 主视觉图
    ├── img2_features.html    # 功能特点图
    ├── img3_ai_review.html   # 多代理评审图
    └── img4_cta.html         # 行动号召图
```

## 使用方式

### 1. 文案

1. 双击打开 `copywriting.html`
2. 点击"复制"按钮复制单个部分
3. 或点击"一键复制全部文案"复制完整内容
4. 粘贴到微信公众号编辑器

### 2. 封面图

#### 方法 1: 在线转换（推荐）

1. 打开 https://convertio.co/svg-png/
2. 上传 `cover.svg`
3. 下载 PNG 文件（900x383 像素）

#### 方法 2: 截图

1. 双击打开 `cover.svg`
2. 按 `Win + Shift + S` 截图
3. 粘贴到画图工具，保存为 PNG

### 3. 宣传图片

#### 方法 1: 浏览器截图（推荐）

1. 双击打开 HTML 文件（如 `img1_hero.html`）
2. 按 `F12` 打开开发者工具
3. 按 `Ctrl + Shift + P` 打开命令面板
4. 输入 `screenshot` 选择 "Capture node screenshot"
5. 保存为 PNG

#### 方法 2: 在线转换

1. 打开 https://htmlcsstoimage.com/
2. 粘贴 HTML 代码
3. 设置尺寸 900x500
4. 下载 PNG
